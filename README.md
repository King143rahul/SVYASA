# Svyasa Secrets

A safe and anonymous platform for students to share their confessions. All posts are completely anonymous and automatically expire after 48 hours.

## Features

- **Anonymous Posting**: Share your thoughts without any personal identification
- **Department & Year Selection**: Specify your academic background when posting
- **Hashtag Support**: Categorize confessions with hashtags
- **Auto-Expiration**: Posts automatically disappear after 48 hours
- **Comment system**: Engage with others' confessions anonymously
- **Modern UI**: Built with React, TypeScript, and Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd svyasaconfession-main/svyasaconfession-main
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:8080](http://localhost:8080) in your browser

## Technologies Used

- **Frontend**: React 18, TypeScript, Vite
- **UI Framework**: shadcn/ui with Radix UI components
- **Styling**: Tailwind CSS
- **Backend**: Supabase (for data storage and authentication)
